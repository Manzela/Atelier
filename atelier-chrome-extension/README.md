# Atelier Chrome Extension

Right-click any web page → "Redesign with Atelier". Captures the rendered DOM, sends to Atelier's convergence loop, returns a side-by-side comparison.

## Status

**Phase 0** — Manifest V3 scaffolded; implementation is a Phase 3 deliverable (Jun 1).

Will be submitted to Chrome Web Store at v1.0.0 release on 2026-06-03.

## Features (planned)

- **Right-click → Redesign** — captures rendered DOM + computed styles + screenshots
- **Side-by-side comparison** — original vs converged with per-axis improvement deltas
- **Local design system import** — extracts brand colors + typography from the visited page
