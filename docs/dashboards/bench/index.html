How to read this

Atelier is gate-first: each design candidate must clear a fixed sequence of quality gates, and a candidate only converges once its composite score reaches ≥ 0.70. Every metric below is shown against that bar or its stated target, so you can see what good looks like next to the actual result — not a number in isolation.

Atelier vs a single-shot baseline

Same briefs, same gate path. The baseline emits one design with no iteration; Atelier runs the full gate-first pipeline. This isolates the value of the pipeline itself.

verifiedDefinition of good

The quality gates

These gates define what “good” means for a candidate. They run fail-closed and in order; a candidate that fails any one of them is rejected. On top of the deterministic gates, a D‑O‑R‑A‑V consensus (Design, Originality, Relevance, Accessibility, Visual‑clarity) scores each surviving candidate to produce the composite used for the 0.70 bar.
