# WebGen-Bench 50-Task Evaluation Subset

## Source

- **Benchmark:** [WebGen-Bench](https://github.com/mnluzimu/WebGen-Bench) (NeurIPS 2025)
- **Full dataset:** 101 tasks, 647 test cases
- **Subset:** 50 tasks, deterministically selected

## Task ID Selection

The 50-task subset is generated by:

1. Loading all 101 task IDs from `data/test.jsonl` in the upstream repo
2. Sorting by `SHA-256(task_id)` — deterministic, reproducible
3. Taking the first 50

To regenerate:

```python
import json, hashlib

tasks = []
with open("data/test.jsonl") as f:
    for line in f:
        tasks.append(json.loads(line)["id"])

subset = sorted(tasks, key=lambda t: hashlib.sha256(t.encode()).hexdigest())[:50]
```
