# Atelier Figma Plugin

Pull Atelier converged designs into Figma as components. Generate new designs from a brief without leaving Figma.

## Status

**Phase 0** — `manifest.json` scaffolded; implementation (TypeScript bridge + UI) is a Phase 3 deliverable (Jun 1).

Will be submitted to Figma Community at v1.0.0 release on 2026-06-03.

## Features (planned)

- **Generate from brief** — open Atelier's PIP intake from inside Figma; converged result lands as a Figma frame
- **Audit current selection** — select a frame; Atelier evaluates against D-O-R-A-V judges; diagnostics overlay
- **Pull from Atelier project** — sync converged designs from an Atelier project; updates as components
